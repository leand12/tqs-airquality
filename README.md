### How to run locally:

#### Frontend
```
$ npm i

$ npm start
```
To make it available on: [http://localhost:3000](http://localhost:3000)


#### REST API
```
$ mvn install

$ mvn spring-boot:run
```
To make it available on: [http://localhost:8080/api/v1](http://localhost:8080/api/v1)
